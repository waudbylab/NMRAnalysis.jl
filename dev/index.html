NMRAnalysis.jl

NMRAnalysis.jl is a Julia package for interactive analysis of biomolecular NMR experiments, with a particular focus on relaxation, diffusion, exchange, and protein dynamics measurements. It provides both 1D command-line workflows and 2D interactive graphical interfaces, and is designed to make common NMR analyses straightforward without sacrificing flexibility.

Active development

NMRAnalysis.jl is under active development. Features and API may change as the package evolves.

What's available

1D Analyses

Command-line tools for routine 1D experiments:

FunctionExperiment
diffusion()DOSY / diffusion coefficient measurement
relaxation()R1 and R2 relaxation (exponential or inversion-recovery fit)
tract()TRACT experiment for rotational correlation time
r1rho()¹⁹F R1ρ relaxation dispersion

2D Analyses

Interactive graphical interfaces for 2D and pseudo-3D experiments:

FunctionExperiment
fit2d()Peak fitting: positions, linewidths, and amplitudes
relaxation2d()R1 / R2 relaxation from a series of 2D spectra
recovery2d()Inversion or saturation recovery
modelfit2d()Custom model fitting
hetnoe2d()Heteronuclear NOE
cpmg2d()CPMG relaxation dispersion
cest2d()CEST (Chemical Exchange Saturation Transfer)
pre2d()Paramagnetic relaxation enhancement
ccr2d()Cross-correlated relaxation

All 2D functions share the same interactive GUI — see the 2D Overview for a guide to the interface and common workflow.

Ecosystem

NMRAnalysis.jl is part of a suite of Julia packages for NMR data handling developed by the Waudby lab:

  • NMRTools.jl — the foundation for NMR data import and processing in Julia. NMRAnalysis.jl is built on top of NMRTools for all data loading, axis handling, and spectral processing.

  • NMRScreen.jl — tools for fragment and ligand screening by NMR, including automated analysis of large compound libraries.

Contributing

NMRAnalysis.jl is developed and maintained by the Waudby lab at University College London. Contributions are warmly welcomed — whether that's bug reports, new analysis routines, documentation improvements, or example datasets. Please open an issue or pull request on GitHub.
